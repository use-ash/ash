# Web Research Skill

## Purpose

Use this skill when a task requires current external information or source-backed research.

## When To Use This Skill

- the answer may have changed recently
- the task requires source links
- the question depends on current events, products, or policies

## Inputs

- research question
- recency requirement
- source quality requirement
- output format

## Procedure

1. Read this file.
2. Read `feedback.log`.
3. Gather the research question and constraints.
4. Search high-quality sources.
5. Compare findings and capture uncertainty.
6. Return a concise answer with sources.

## Output

Return:

- a concise answer
- source links
- uncertainty notes if the evidence is incomplete or conflicting

## Failure Modes

- relying on stale sources
- omitting source attribution
- presenting inference as confirmed fact

## Feedback Application

Apply the feedback log before starting the research process.
