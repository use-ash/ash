# use-ash/skills

Framework for building reusable agent capabilities as self-contained skill directories.

This repository describes how to package a capability so an agent can recognize when to use it, read its local instructions, apply stored feedback, and improve over time.

This repository contains no executable code. It contains templates, structure guides, and examples.

Start here:

1. Read [AGENT_INSTRUCTIONS.md](AGENT_INSTRUCTIONS.md).
2. Read [SKILLS_FRAMEWORK.md](SKILLS_FRAMEWORK.md).
3. Read [SKILL_TEMPLATE.md](SKILL_TEMPLATE.md).
4. Read [FEEDBACK_LOG_SPEC.md](FEEDBACK_LOG_SPEC.md).
5. Read [CREATING_A_SKILL.md](CREATING_A_SKILL.md).
